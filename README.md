# 适用于争渡读屏的文星盲文显示器插件

## 功能特性

- 读屏的盲文显示；
- 路由光标键和其他硬件按钮的识别并上报给读屏；

## 安装方法

首先，应当确保盲文显示器设备的驱动程序已经安装配置完成。

其次，安装争渡读屏文星盲文显示器插件。
。要安装插件，请将该文件夹下的 “StarBrailleDisplay” 文件夹复制到读屏根文件夹下的 “addins\BrailleDisplay” 文件夹下。
如果是更新插件，需要退出读屏才能完成替换更新操作。

## 按键布局

目前支持文星盲文电脑，按键布局分别介绍如下。

### 文星盲文电脑按键布局

盲文点阵的每一方上对应相应路由光标键，按下使编辑光标跳转到对应位置。

机器下部一共有六个按键。左侧和右侧分别有一组两个按键，中间有两个单独的按键。

从左到右依次为：

- 上一行；
- 上一屏；
- Tab 键；
- 回车键；
- 下一屏；
- 下一行。
